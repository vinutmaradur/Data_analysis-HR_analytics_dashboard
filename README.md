# HR Analytics Dashboard 📊

# Project Overview 🔍
This project is an interactive HR Analytics Dashboard built using Tableau to analyze employee data and uncover key workforce insights. The dashboard helps HR teams and decision-makers monitor trends in employee attrition, performance, demographics, and overall workforce health.

# Objectives 🎯
- Analyze employee attrition patterns
- Identify factors affecting employee retention
- Track department-wise performance
- Understand demographic distribution
- Enable data-driven HR decisions

---

# Dataset Description 📁
The dataset includes the following key fields:

- Employee ID
- Age
- Gender
- Department
- Job Role
- Salary / Income
- Years at Company
- Attrition (Yes/No)
- Performance Rating
- Education Level
- Job Satisfaction

---

# Dashboard Features 📊
1. Attrition Analysis
- Overall attrition rate
- Attrition by department and job role
- Attrition by age group and experience
2. Employee Demographics
- Gender distribution
- Age distribution
- Education level breakdown
3. Performance Insights
- Average performance rating
- Performance by department
- Correlation between satisfaction and performance
4. Salary & Experience Trends
- Salary distribution across roles
- Experience vs attrition patterns
- Income vs job satisfaction
5. Interactive Filters
- Department
- Job Role
- Gender
- Education
- Years of Experience
